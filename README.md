# 概要
趣味でUnityやUnreal Engineの勉強を目的に制作しているものの置き場です。

※ Space_ShooterおよびDrArwardは、他の開発者の方が公開されている著作物であり、それらを参考に制作したものです。素材も公開されているものを使用しております。

※UnityのVersionについて、Card_Gameは2020.3.20f1、Space_Shooterは2020.3.28f1です。  
※Unreal EngineのVersionについて、DrAwardは4.27.1です。

各作品についての詳細は、Portfolio/portfolio.pdf をご覧ください。  
操作方法は、各制作物のディレクトリ直下にある 操作方法.txt をご覧ください。

ソースコードにつきましては、以下ディレクトリ内をご参照ください。（※Unreal EngineのBlue Printの閲覧は専用のUnreal Editorが必要になります）
・Space_Shooter：Space_Shooter/Assets/Scripts/
・DrAward：DrAward/Content/BluePrint/
・Card_Game：Card_Game/Assets/Scripts/, Card_Game/Assets/CardDatas/
